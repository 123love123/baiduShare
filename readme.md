# 百度分享不支持Https的解决方案  #
### 使用方法： ###
将其static文件夹放在网站的根目录下，并将对应的百度分享代码中，把http://bdimg.share.baidu.com/改为 /
### 如何修改获得该代码等详见Blog： ###
https://www.hrwhisper.me/baidu-share-not-support-https-solution/
